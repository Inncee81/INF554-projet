EXPLICATION:

   De Windows 8.1 est le nom de la dernière version de Microsoft Windows,
   la version grand public de qui sera lancé le 18 Octobre 2013. Epson
   s'est engagé à soutenir Windows 8 et de Windows 8.1 et ne cesse de
   développer des applications et des pilotes de périphériques à travers
   notre gamme de produits d'imagerie numérique pour se conformer aux
   nouvelles normes mondiales de Microsoft.

   Windows 8 et Windows 8.1 prend en charge certaines produits Epson dès
   la sortie de la boîte, en plus de ce que nous avons publié une série de
   mises à jour de pilotes pour correspondre avec le lancement de Windows
   8. Voir les tableaux ci-dessous et si la demande est de télécharger le
   driver s'il vous plaît visitez le site Web d'Epson.

Comment Désinstaller une imprimante avec Windows 8

CONCERNE:

     * Désinstaller le pilote d’impression Epson Printer
     * Gamme Epson printer
     * Windows 8, Windows 8 x64

TABLE DES MATIÈRES:

Explication

EXPLICATION:

   Suivez les étapes suivantes pour désinstaller le pilot d’impression
   depuis Windows 8.
    1. Commencez par mettre sous tension l’imprimante et ensuite
       connectez-la à l’ordinateur.
    2. Ouvrez le dossier Périphériques et Imprimantes. Vérifiez qu’il n’y
       ait aucun document en attente d’impression pour toutes les
       imprimantes Epson installées, le ou les supprimer dans le cas
       contraire.

                                       Remarque:
   Si vous avez plus d’une icône pour votre modèle d’imprimante Epson,
   répétez la procédure pour tous les pilotes dupliqués.
    3. Eteignez l’imprimante et déconnectez le cable data.
    4. Pour désinstaller le pilote de l’imprimante, faites bouger le
       curseur de la souris en bas ou au coin supérieur droit de votre
       écran et cliquer sur l’icône Rechercher (comme indiqué sur l’image
       ci-dessous) depuis le menu qui apparait.
    5. Taper Désinstaller dans la boite de recherche et sélectionner
       Désinstaller le logiciel Epson Printer EPSON.
          + Alternativement vous pouvez désinstaller l’imprimante en
            allant dans le menu Panneau de Configuration, Programmes.
            Sélectionner l’imprimante à désinstaller dans la liste Ex :
            Désinstaller EPSON WF-2520 Séries et cliquer sur
            Désinstaller/Changer.
          + Cliquer sur votre imprimante comme à l’écran et cliquer sur OK
            pour désinstaller.

                                      Remarque:
   Cliquer sur Continuer si vous voyez apparaitre une fenêtre Windows vous
   demandant la permission de "Désinstaller ou changer l’application".
    6. Cliquer Oui lorsqu’il vous est demandé si vous souhaitez confirmer
       la suppression de l’imprimante sélectionnée:
    7. Vous verrez ensuite la confirmation que votre imprimante a bien été
       supprimé et qu’une autre imprimante a été configuré par défaut. Si
       aucune autre imprimante n’est installée, ce sera habituellement le
       'Microsoft XPS Document Writer' qui sera configuré.
    8. Vous verrez ensuite la confirmation que la désinstallation est
       terminée, cliquer OK.
    9. Assurez-vous que l’imprimante est toujours hors tension et
       déconnectée de l’ordinateur. Ensuite, redémarrez votre ordinateur
       et déplacez le curseur de la souris en bas ou au coin supérieur
       droit de votre écran et cliquer sur Paramètres. Cliquer sur Allumez
       puis Redémarrer.
   10. Une fois l’ordinateur redémarré, la désinstallation est terminée.
          + Si vous avez besoin de réinstaller le pilote de l’imprimante,
            faites-le en utilisant le CD-ROM original ou la dernière
            version recommandée depuis le site Epson Support.
          + Pour plus d’informations regarder la rubrique Epson and
            Windows 8.

Procédure de désinstallation d'une imprimante sous Windows 7

CONCERNE:

     * Windows 7

TABLE DES MATIÈRES:

Explication

EXPLICATION:

   Procédez comme suit pour désinstaller le pilote d'impression sous
   Windows 7.
    1. Commencez avec l'imprimante sous tension et connectée à
       l'ordinateur.
    2. Ouvrez le dossier Printers (Imprimantes). Identifiez et annulez les
       documents en attente d'impression pour toutes les imprimantes Epson
       installées.

                                [Notes.gif]  Remarque:
   Si plusieurs icônes s'affichent pour votre ou vos modèles d'imprimante
   Epson, répétez cette procédure pour les pilotes en double.
    3. Mettez l'imprimante hors tension et déconnectez le câble de
       données.
    4. Pour désinstaller le pilote d'impression, cliquez sur l'icône Start
       (Démarrer), généralement située dans le coin inférieur gauche de
       l'écran, puis sélectionnez Epson Printer Software Uninstall
       (Désinstallation du logiciel d'impression Epson).
       [Uninstall_box_out.png]

                           [Notes.gif]  Remarque:
   Cliquez sur Continue (Continuer) si Windows affiche une demande
   d'autorisation de type Uninstall or change an application (Désinstaller
   ou modifier une application).
    5. Cliquez sur Yes (Oui) lorsque vous êtes invité à confirmer que vous
       souhaitez supprimer l'imprimante sélectionnée:
       [Delete_Confirm.png]
    6. Le message ci-dessous s'affiche alors. Cliquez sur Yes (Oui).
       [Some_Files.png]
    7. La confirmation de la suppression de l'imprimante et de la
       sélection d'une autre imprimante par défaut s'affiche ensuite. Si
       aucune autre imprimante n'est installée, l'imprimante par défaut
       est généralement l'imprimante Microsoft XPS Document Writer.
       [Personal_Settings.png]
    8. La confirmation de la suppression de l'imprimante et de la
       sélection d'une autre imprimante par défaut s'affiche ensuite. Si
       aucune autre imprimante n'est installée, l'imprimante par défaut
       est généralement l'imprimante Microsoft XPS Document Writer.
       [Doc_Writer.png]
    9. La confirmation de la finalisation de la désinstallation s'affiche.
       Cliquez sur OK.
       [Uninstall_Complete.png]
   10. Assurez-vous que l'imprimante est toujours hors tension et
       déconnectée de l'ordinateur. Redémarrez ensuite l'ordinateur en
       cliquant sur Start (Démarrer), puis sur la flèche située au bas du
       menu, puis sur Shut Down (Arrêter).
   11. Le processus de désinstallation est terminé au redémarrage de
       l'ordinateur.
          + Si vous devez réinstaller le pilote d'impression, utilisez le
            CD-ROM d'origine (si le produit est sorti après la mi-2007) ou
            les derniers pilotes recommandés du site Web d'assistance de
            Epson.
          + Pour plus d'informations, reportez-vous à l'article Epson et
            Windows 7, dans la section des articles associés.

Mac OS X: Comment puis-je installer un pilote d'imprimante EPSON (papier en
rouleau/sans bordure)?

CONCERNE:

     * Imprimantes à jet d'encre
     * Imprimantes grand format

TABLE DES MATIÈRES:

Explication

EXPLICATION:

   Cet article explique comment installer une imprimante à jet d'encre
   EPSON sous Mac OS X.

   *Dépend de la langue utilisée sur votre Macintosh et l'imprimante.

   Si vous procédez à l'installation à partir du CD, vous pouvez voir
   l'écran suivant après avoir inséré le CD dans le lecteur correspondant
   et double-cliqué sur le dossier Mac OS X : cet écran dépend de la
   langue utilisée sur votre Macintosh et l'imprimante. Seuls sont inclus
   les CD-ROM anglais, français et italien (existent également les
   versions allemande, néerlandaise et espagnole.).

   [OSX-Install-CD-Driver_folder.png]

   Dans le dossier (français, anglais et italien) vous pouvez trouver des
   logiciels (comme PhotoQuicker, Photostarter.). Ce logiciel est
   disponible pour l'imprimante Stylus Photo R800. Pour une autre
   imprimante, vous aurez peut-être un logiciel différent.

   Pour installer le pilote, ouvrez le dossier Printer driver et
   double-cliquez sur le pilote d'impression (exemple SPR800_18b*).

   Vous devez disposer de droits administrateur pour installer le pilote:

   [OSX-Authenticate-R800.png]

   [OSX-Install-Licence_Agreement.png]

   Après avoir accepté l'accord de licence du logiciel, l'écran suivant
   apparaît:

   [OSX-Install-R800-Easy_Install-s.png]

   Sélectionnez Installation, puis cliquez sur le bouton Install*.
   L'installation commence. Une fois l'installation terminée, vous devez
   redémarrer votre Macintosh. A présent, vous devez installer et
   configurer l'imprimante. Pour ce faire, vous devez lancer un logiciel
   appelé Print Center* (Mac OS 10.1x et 10.2.x) ou Printer Setup Utility*
   (Mac OS 10.3.x et 10.4.x). Ce logiciel Apple est disponible dans le
   dossier /Applications/Utilities/*:

   [OSX-Utilities-Printer_Setup_Utility.png]

   Lancez ce logiciel, puis cliquez sur le bouton Add*. (10.4.x: Cliquez
   sur More Printers, puis cliquez sur le bouton Add.)

   [OSX-Printer_List-Add.png]

   Sélectionnez la connexion utilisée* (comme EPSON AppleTalk, EPSON
   FireWire, EPSON TCP/IP):

   [OSX-Add_Printer-select_interface.png]

   Si l'imprimante* prend en charge le papier en rouleau sans bordure,
   vous devez le définir ici. Cliquez sur le bouton All:

   [OSX-R800-Add-Page_Setup-All.png]

   Cliquez sur Add. Vous pouvez désormais imprimer*:

   [OSX-R800_in_Printer_List_All-s.png]
