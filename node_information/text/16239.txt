Activer l'amélioration automatique dans le logiciel Saal Design

   Vous avez la possibilité d'utiliser l'amélioration automatique pour
   toutes les photos placées dans notre logiciel.

   Afin de l'activer, vous devez placer une photo dans le logiciel. Dés
   que l'image est placée, l'option "amélioration automatique" apparait
   dans la barre des tâches en haut de l'écran :

   Activer l'amélioration automatique dans le logiciel Saal Design

   Après avoir cliqué sur la case et que l'encoche apparait,
   l'amélioration automatique est activée.

   Il est à noter que l'amélioration automatique est activée par défaut
   pour toutes les images qui n'ont pas été retouchées.

   Vous pouvez également choisir si l'amélioration automatique doit être
   utilisée pour toutes les images de votre projet ou seulement sur
   l'image sélectionnée.

Lignes blanches dans fichiers PDF dans l'affichage à 100% du téléchargement
direct

   Commande avec fichier PDF dans le téléchargement direct

   Il arrive malheureusement parfois que des lignes blanches apparaissent
   sur les fichiers PDF lorsque l'affichage de notre téléchargement direct
   est réglé à 100%, voire que ces lignes apparaissent dans des textes ou
   images avec des ombres.

   Lorsque l'on télécharge le fichier PDF final dans le téléchargement
   direct et que l'affichage est réglé à 100%, on peut déjà voir ces
   lignes. Ces lignes seront également visibles lors du développement
   photo.

   Lignes blanches dans fichiers PDF dans l'affichage à 100% du
   téléchargement direct

   Exemple :

   Lignes blanches dans fichiers PDF dans l'affichage à 100% du
   téléchargement direct

   Cause :

   Les lignes sont causées par la réduction de la transparence du fichier
   PDF de InDesign vers Adobe Acrobat ou par l'optimisation dans Adobe
   Acrobat. Ceci est un bug.

   Solution :

   Si des lignes blanches apparaissent lors de l'optimisation du PDF et
   qu'elles sont également visibles dans notre téléchargement direct avec
   un affichage à 100%, la solution est d'exporter d'InDesign les pages au
   format JPG.

   Vous pouvez exporter d'InDesign les pages au format JPG. Veuillez en
   premier lieu vérifier dans nos infos pro quelle résolution vous aurez
   besoin plus tard pour vos données (300 ou 301dpi) et exportez les pages
   d'InDesign suivant les réglages appropriés.

Chemin d'accès des projets dans le logiciel Saal Design

   Logiciel Saal Design :
   Windows Vista und 7 :
   C:\\Users\\<username>\\AppData\\Roaming\\SaalDesignSoftware\\Local
   Store\\projects
   Windows XP:
   C:\\Documents and Settings\\<username>\\Application
   Data\\SaalDesignSoftware\\Local Store\\projects
   Mac OS X:
   /Users/<username>/Library/Application Support/SaalDesignSoftware/Local
   Store/projects

   Saal Design Software Beta:

   Windows Vista und 7:
   C:\\Users\\<username>\\AppData\\Roaming\\SaalDesignSoftwareBeta\\Local
   Store\\projects
   Windows XP:
   C:\\Documents and Settings\\<username>\\Application
   Data\\SaalDesignSoftwareBeta\\Local Store\\projects
   Mac OS X:
   /Users/<username>/Library/Application
   Support/SaalDesignSoftwareBeta/Local Store/projects

   Les chemins d'accès cités plus haut sont les réglages standards. Il est
   cependant possible de les changer dans le logiciel, en cliquant sur la
   petite roue dentée des réglages. Vous trouvez à cet endroit les chemins
   d'accès actuels.

   Sauvegarde des projets :
   Si vous souhaitez sauvegarder vos projets, nous vous recommandons
   d'exporter le projet depuis le logiciel, car dans les projets en
   fichiers sswp, les données image ne sont pas incluses.
