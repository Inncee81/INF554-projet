Comment trouver le code article d’un produit IGN?

   Le code article d’un produit IGN correspond au numéro de la carte. Vous
   retrouverez le numéro de la carte sur sa couverture recto en haut à
   droite et/ou sur la tranche de la carte.

   Pour les séries suivantes, le code article est à composer de la manière
   suivante :
     * Carte régionale : REG+ n° de la carte
       soit pour la carte régionale du Limousin REG12
     * Carte TOP100 : TOP100 + n° de la carte
       soit pour la carte Limoges Guéret TOP100147
     * Carte nationale : 1M + n° de la carte
       soit pour la carte routière France nationale 1M901
     * Carte départementale * : 7211 ou D7212 + n° de la carte/du
       département
       (voir les titres dans l'onglet produits IGN, imagette cartes
       routières)
       soit pour la carte départementale de la Haute Vienne D721287

   * Exceptions :
     * carte départementale Paris et couronne (départements
       75/78/91/92/93/94/95) code article = D721299
     * carte départementale Corse (départements 2A/2B) code article =
       D721220
     * carte départementale Territoire de Belfort (départements 90) code
       article = D721268
