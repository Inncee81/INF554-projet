Foire Aux Questions

   Vous rencontrez une difficultée ? Avant de nous contacter vérifiez que
   la solution ne se trouve pas dans cette page.

Comment s'inscrire à votre service de redirections ?

   Inscrivez-vous en indiquant votre première redirection web (par la
   suite vous pourrez en créer d'autres avec ce même compte), ainsi que
   les informations tel que votre adresse électronique pour la validation
   de votre compte. Une fois votre compte validé, votre redirection sera
   immédiatement activée.
   Ensuite, rendez-vous sur votre panel d'administration (informations
   indiquées dans le mél de validation) et administrez vos redirections.

Je n'arrive pas à valider mon compte via le mél envoyé, comment faire ?

   Il est possible que l'interface de lecture que vous utilisez pour lire
   vos méls ne convertisse pas les liens hypertextes en liens cliquables.
   Il vous suffit alors d'effectuer un "copier/coller" de l'adresse de
   validation dans votre navigateur internet favoris.
   Si vous obtenez une erreur, signalez le nous via la page de Contact.

Je n'arrive pas à retenir le mot de passe attribué, comment le modifier ?

   Il suffit de vous rendre sur votre compte et d'aller dans la rubrique
   "Mes infos", ainsi vous pourrez indiquer un nouveau mot de passe.

Le domaine que j'ai ajouté n'est pas redirigé vers le bon site ou ne
fonctionne pas !

   Vérifiez que l'adresse (URI) d'origine de votre site est correctement
   indiquée (rubrique "Redirections WEB"), celle-ci doit être l'adresse
   actuelle de votre site web. Si besoin est, éditez le domaine en
   question.
   Faites également attention à correctement orthographier l'extension de
   votre nom de domaine.
   Par exemple, les extensions 0rg.fr, ass0.fr, c0m.at contiennent le
   chiffre zéro et non la lettre O.

J'ai perdu mon mot de passe, que faire ?

   Vous disposez d'un bouton de récupération de votre mot de passe
   disponible dans le formulaire d'identification "Espace membre", à
   droite du bouton de validation "Ok". Vous pouvez également y accéder
   via ce lien.

Redirection gratuite Transparente (cloaking)

   Ce mode de redirection, appelé aussi redirection cloakée, permet
   d'afficher votre site à vos visiteurs tout en gardant l'adresse de
   redirectionnement (votre nom de domaine) visible dans la barre de
   saisie d'adresse du navigateur.
   La redirection est dite "transparente", car le visiteur ne voit pas de
   changement d'URL, ni la vrai adresse de l'hébergement lors de la
   consultation du site.

Obtenez dès maintenant votre nom de domaine gratuit avec redirection gratuite
transparente !
