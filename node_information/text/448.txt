   Pas à pas pour la version à télécharger :

   1) Créer un dossier sur votre ordinateur (ex: didapages).

   2) Faire un clic-droit sur un lien ci-dessus puis "Enregistrer le lien
   (la cible du lien) sous..." et enregistrer le fichier dans le dossier.

   3) Dans le dossier, faire un clic-droit sur le fichier, choisir
   "Extraire tout..." puis Extraire.

   4) Dans le dossier qui vient de s'ouvrir, double-cliquez sur le fichier
   "didapage" avec un numéro.
