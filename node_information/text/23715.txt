AffiliateWP – Plugin d’Affiliation WordPress

     * avril 12, 2017

     Aujourd’hui, je vous présente AffiliateWP, le meilleur plugin
     d’affiliation sur WordPress pour les annonceurs. Sur de nombreux
     sites Internet des liens commerciaux ou des bannières publicitaires
     sont affichés. Que cela soit dans le header, dans un article / page
     ou dans la sidebar, ces publicités ont un but bien précis : inciter
     les internautes à acheter le produit présenté.

   Ce système d’affichage de publicité est souvent mis en place sous forme
   d’affiliation. L’affiliation est connue par tous les webmasters. D’un
   côté il y a un annonceur et un diffuseur. En clair, l’annonceur va
   autoriser le diffuseur à afficher une publicité de ses produits ou ses
   services sur son site. En contrepartie, tous les visiteurs cliquant sur
   la bannière publicitaire permettront au diffuseur de toucher une
   commission au moment même où une conversion est effectuée. Son succès
   est grandissant depuis quelques années et le développement de sites
   spécialisés ou d’extensions spécialisées n’en finit plus.

Présentation du plugin AffiliateWP

   Ce plugin marketing d’affiliation connaît un fort succès. Si vous êtes
   un annonceur et que vous souhaitez commencer un programme d’affiliation
   pour booster votre chiffre d’affaires et vos visites, AffiliateWP est
   le plugin qu’il vous faut. Simple d’utilisation, en l’installant sur
   votre site WordPress, vous aurez la possibilité de proposer des liens
   d’affiliation à d’autres webmasters. Grâce à ces derniers, vous aurez
   l’occasion de gagner de nouvelles ventes et d’augmenter votre notoriété
   dans votre secteur d’activité. Un marché gagnant dans les deux sens,
   aussi bien de votre côté en tant qu’annonceur que du côté du diffuseur
   qui gagnera un pourcentage sous forme de commission.

   Le plugin WordPress AffiliateWP a été conçu dans l’idée de répondre à
   la demande des annonceurs et diffuseurs. Les liens affiliés diffusés
   sur les sites auront pour but de faire fructifier les affaires et
   d’apporter un revenu supplémentaire aux diffuseurs. Ce plugin se
   configure rapidement et simplement sur tous les sites WordPress.
   Compatible avec WooCommerce et d’autres extensions de commerces en
   ligne, les professionnels ayant choisis WordPress pour créer leur
   boutique en ligne seront ravis d’utiliser le plugin premium
   AffiliateWP.
     __________________________________________________________________

Les fonctionnalités du plugin AffiliateWP passées au crible

   Une fois que vous aurez connu toutes les fonctionnalités du plugin,
   vous ne pourrez qu’en tomber amoureux. Les développeurs ont gâté les
   webmasters pour cette extension. Les nombreuses fonctionnalités
   présentes vont permettre aux annonceurs d’avoir un suivi de leurs
   ventes en fonction des diffuseurs, ainsi qu’un suivi des meilleures
   publicités. Des fonctionnalités importantes qui marquent le succès du
   plugin.

   AffiliateWP s’installe et s’active comme toutes les autres extensions.
   Une fois activée, une page de bienvenue s’affiche et vous donne des
   détails sur les personnes ayant aidé à construire ce plugin. Cet onglet
   vous propose aussi de commencer les premiers paramétrages.

Tableau de bord

   Le tableau de bord vous donne des informations utiles concernant les
   différentes statistiques de vente de vos produits par le biais des
   affiliations. Ces informations se regroupent sous différents labels.
   Vous avez les statistiques globales, combien avez-vous dépensé pour
   payer vos affiliés et quels sont les montants dus. Vous retrouvez
   également les derniers affiliés enregistrés ou ceux vous rapportant le
   plus de ventes.

   Ces statistiques vous permettent de redéfinir vos axes en fonction des
   données et de privilégier l’un ou l’autre de vos affiliés. Mais passons
   dans le vif du sujet avec, notamment, la configuration du plugin.
   AffiliateWP WordPress - Tableau de Bord AffiliateWP WordPress – Tableau
   de Bord

Les différents onglets du plugin

   Chaque onglet propose des données uniques et indispensables pour bien
   faire fonctionner le plugin premium d’affiliation WordPress. En
   configurant comme il se doit l’extension, vous mettez toutes les
   chances de votre côté d’augmenter vos revenus et vos visites.

Affiliate

   Sur cet onglet, vous avez un aperçu de tous les affiliés de votre site.
   Vous visualisez ceux qui sont actifs, inactifs, en attente et ceux que
   vous avez rejetés. Les colonnes du tableau montrent les statistiques
   rapides de chaque affilié. Combien ont-ils gagné, combien de visites
   avez-vous reçues grâce à eux, etc. Un onglet spécifique est aussi mis
   en place pour les gérer et en ajouter manuellement.

   Pour ajouter un affilié, cliquez sur le bouton « Add new ». Choisissez
   dans la liste déroulante l’affilié que vous souhaitez inscrire sur
   votre site manuellement et remplissez les différents champs : user,
   affiliate status, referral rate type, referral rate (quel pourcentage
   de commission pour cet utilisateur), payment email (pour qu’il soit
   payé), welcome email.

   Pour éditer un affilié, cliquez sur le bouton « Edit ». Vous avez la
   possibilité de changer son pourcentage de commissionnement, l’adresse
   email pour le paiement et l’email du compte. Le reste n’est pas
   disponible à l’édition.

   Enfin, dans le menu « Affiliates », vous avez également la possibilité
   de voir des statistiques sur les derniers jours, dernières semaines et
   derniers mois. Une courbe des gains s’affichera en fonction des
   utilisateurs. Vous découvrirez ainsi quels sont vos meilleurs
   diffuseurs.
   AffiliateWP WordPress - Affiliates AffiliateWP WordPress – Affiliates

Referrals

   Cette page concerne le total des références par utilisateur. En un coup
   d’oeil, vous découvrez les paiements en attente. Sur chaque ligne, vous
   pouvez marquer une transaction comme payée, rejetée ou acceptée. Il
   vous est aussi possible de modifier ou supprimer une transaction. Là
   encore vous pouvez ajouter manuellement une référence et éditer une
   existante. Une option de génération de fichier CSV vous permet de créer
   un fichier regroupant toutes les ventes dans une période précise.
   AffiliateWP WordPress - Referrals AffiliateWP WordPress – Referrals

Visitors

   Cette page recense toutes les visites effectuées par le biais de vos
   affiliés. Des données importantes pour connaître d’où proviennent les
   revenus engendrés sur votre site. Le programme d’affiliation pourra
   ainsi être bénéfique pour vous, car vous verrez les conversions
   abandonnées et celles qui ont eu lieu. Adresse ip, date et heure de la
   visite, provenance de la visite et landing page sont les principales
   informations diffusées.

Creatives

   Pour proposer à vos affiliés une publicité qui se fond dans la masse,
   l’onglet « Creatives » est celui qu’il vous faut. Grâce aux visuels
   proposés, les affiliés pourront booster leurs ventes et attirer encore
   plus de visiteurs vers votre site. Une manière simple de générer
   davantage de trafic et de chiffre d’affaires.

   Pour ajouter un nouveau visuel, cliquez sur « Add new ». Les champs
   suivants seront à remplir : name, description, url, text, image et
   status. Une fois ces éléments mis en place, la bannière publicitaire
   sera disponible pour vos utilisateurs. Ils recevront un shortcode à
   insérer dans leur page.

   Dans le mode édition, vous retrouvez les mêmes champs que pour l’ajout.

Reports

   L’onglet reports apporte les informations nécessaires pour connaître
   les statistiques de son site par rapport à ses affiliés. L’onglet
   visite vous laisse découvrir le montant total des visites, le taux de
   conversion par affilié et les commissions en cours. Quelle est votre
   meilleure campagne ou quel est votre meilleur affilié ? L’outil vous le
   dira en se basant sur les dernières transactions.
   AffiliateWP WordPress - Reports AffiliateWP WordPress – Reports

Tools

   Gardez en tête que vous devez exporter régulièrement vos données, car
   vous n’êtes pas à l’abri d’un hack de votre site ou d’un bug. Pour
   cela, l’onglet « Tools » sera parfait pour vous. Il vous permet
   d’exporter des fichiers CSV pour les données des affiliés, des
   références, mais aussi sur la configuration actuelle du plugin. Cela
   vous permettra ensuite d’importer facilement votre configuration si
   votre site venait à rencontrer un problème et que vous perdiez tout.

Settings

   Certains paramètres peuvent être bricolés selon vos envies et vos
   besoins. L’onglet est donc prévu à cet effet. Dans un premier temps, il
   vous faudra renseigner la clé de votre licence et l’activer pour que le
   plugin fonctionne. Ensuite, vous devez renseigner les pages
   importantes, comme celles où les affiliés peuvent manager leur compte
   et celles expliquant les termes d’utilisation.

   Les autres paramètres concernent le pourcentage (ou le montant) de
   commissionnement, la devise (euro, dollar, etc.), l’expiration des
   cookies… La liste des endroits où doit être intégré AffiliateWP est à
   configurer dans cet onglet. Les emails envoyés automatiquement aux
   affiliés sont à définir dans l’onglet des paramètres. Vous pouvez
   changer les messages comme bon vous semble. Des étiquettes de
   personnalisation sont proposées, ce qui rend vos emails plus marquants.

Add-ons

   Des plugins additionnels sont disponibles dans cet onglet. N’hésitez
   pas à les consulter pour compléter votre offre pour vos affiliés.

SEOPress – L’Ultime extension SEO pour WordPress – Gratuit + Version PRO

     * mars 28, 2017

   Sur Geekeries, vous avez pu découvrir le célèbre plugin WordPress SEO
   by Yoast et son alternative The SEO Framework. Je viens de tester un
   nouveau module et je ne pouvais pas passer à côté de sa présentation.
   Il surclasse selon moi les précédents cités. Ergonomique et facile à
   prendre en main, ses innovations techniques et ses fonctionnalités
   complètes font de cette extension WordPress la plus performante pour
   optimiser un site en SEO.

   Il est très compliqué de s’y retrouver avec toutes les extensions SEO,
   que ce soit du côté des gratuites ou payantes. SEOPress répond aux
   problématiques de tous les éditeurs.
   Télécharger SEOPress
     __________________________________________________________________

Présentation du plugin SEOPress

   Vous le savez tous, l’optimisation d’un site Internet est indispensable
   si l’on souhaite le faire monter dans les premiers résultats de Google.
   Tout faire à la main n’est pas évident, voire judicieux et c’est la
   raison pour laquelle des plugins WordPress sont disponibles. Pour
   répondre à la demande des éditeurs d’avoir un plugin complet en matière
   d’optimisation du référencement naturel, Benjamin DENIS a mis les
   bouchées doubles et a développé SEOPress.
   SEOPress SEOPress

   Les témoignages sont unanimes : le plugin freemium surpasse les autres
   plugins SEO. Sa version gratuite est bien conçue et offre tous les
   avantages et toutes les fonctionnalités que les éditeurs recherchent.
   Les inconvénients rencontrés avec WordPress SEO ne sont pas présents
   sur SEOPress. Alors, pourquoi ne pas passer directement sur cette
   extension ?

     Voyons ensemble certaines fonctionnalités gratuites de l’extension.
     __________________________________________________________________

Les fonctionnalités complètes de SEOPress

   Avoir une extension WordPress complète et performante n’est pas
   toujours facile à trouver. Pourtant, les fonctionnalités gratuites
   de SEOPress montrent toute la puissance du plugin. Je vous montre
   aujourd’hui quelques fonctions présentes dès que vous installez le
   plugin.

Tableau de bord

   Depuis le tableau de bord, vous accédez à toutes les options
   disponibles. D’un simple clic, vous activez ou désactivez une fonction.
   Vous pouvez également vous rendre dans l’interface de gestion de chaque
   partie en cliquant sur le bouton « Gérer » disponible dans chaque
   encadré.

Titres et Métas

   Les titres et les métadonnées sont importants en référencement naturel,
   mais je ne vous l’apprends pas. De ce fait, il est normal que ce soit
   la première fonctionnalité à laquelle vous puissiez accéder depuis le
   plugin. Cinq onglets apparaissent, avec chacun des options bien
   précises.
     * Accueil : paramétrez ici la balise title et meta descriptions de
       votre page d’accueil.
     * Détail types de contenus : articles, pages, images, les
       paramétrages se font dans cette partie. Vous pouvez aussi choisir
       de mettre les types de contenus en noindex ou nofollow et
       d’afficher ou non la date dans les résultats de Google.
     * Archives : même chose qu’avant avec l’ajout d’autres types de
       contenus « archives ». Les balises pour la page 404 se modifient à
       cet endroit.
     * Taxonomies : ici ce sont les catégories et les étiquettes qui sont
       les cibles des paramétrages.
     * Avancé : permets de gérer les paramètres de toutes les pages sans
       exception. Si une fonction doit être activée sur toutes les pages,
       cet onglet est là pour vous.
     * Un problème pour retrouver les variables à insérer ? Cliquez sur
       « plus de balises », une aide vous est apportée.

   SEOPress - Titre SEOPress – Titre

   Sitemap XML / HTML

   Le sitemap xml est indispensable si vous souhaitez indexer votre site
   sur Google. C’est lui qui va indiquer aux moteurs de recherche quelles
   pages sont à indexer et quelles sont celles qu’il n’a pas besoin de
   prendre en compte.
     * Général : permets d’activer ou non le sitemap xml et html. Vous
       pouvez également pinger Google directement depuis le plugin. Un
       gain de temps non négligeable.
     * Types de contenu et Taxonomies : ces deux parties vous permettent
       de spécifier quels types de contenus doivent apparaître dans le
       sitemap et lesquels doivent être bloqués.
     * Sitemap HTML : avec le sitemap HTML, vous proposez un plan du site
       cohérent à vos visiteurs. Un boost également pour le SEO à ne pas
       négliger.

   SEOPress - Plan SEOPress – Plan

Social

   Selon le site, vous allez vouloir associer vos comptes sociaux.
   Indispensables pour booster votre notoriété, je ne peux que vous
   conseiller de bien renseigner les comptes dans cette partie (et surtout
   d’activer l’option).
     * Knowledge graph : différents champs à remplir pour configurer le
       Knowledge graph de Google (numéro de téléphone, nom, logo…).
     * Vos comptes sociaux : indiquez dans les champs correspondants les
       urls de vos comptes sociaux et affichez-les directement sur votre
       site.
     * Facebook : pour gérer et configurer l’Open Graph Facebook,
       remplissez les différents champs prévus à cet effet.
     * Twitter : idem que pour Facebook, mais cette fois pour Twitter. Les
       partages sur ces deux réseaux seront ainsi mieux contrôlés.

   SEOPress - Social SEOPress – Social

Avancé

   Chaque extension WordPress a des fonctionnalités avancées. SEOPress ne
   déroge pas à la règle.
     * Avancé : rediriger les pages de pièces jointes, supprimer les mots
       inutiles des urls ou le /category/ ou encore effectuer la
       vérification de la Search Console ? Toutes ces options sont
       disponibles à cet endroit.
     * Apparence : comme tout plugin, il est important de pouvoir gérer
       l’affichage du plugin dans le back-office. Entre la suppression de
       SEOPress dans la barre d’outils, l’ajout ou non du champ titre dans
       un article ou la case noindex dans les articles, différents
       paramètres peuvent être réglés.
     * Sécurité : si vous gérez plusieurs utilisateurs sur votre site,
       vous serez heureux de savoir que vous pouvez leur rendre accès ou
       non aux options SEO du plugin.

SEOPress – Version PRO

   SEOPress PRO SEOPress PRO

   D’autres fonctionnalités très intéressantes sont à découvrir sur le
   plugin.
   On y retrouve les fonctionnalités suivantes :
     * WooCommerce : compatibilité totale avec le plugin WooCommerce qui
       convertit WordPress en un véritable E-Commerce
     * Dublin Core : permet d’utiliser les balises meta Dublin Core
       (dc.title, dc.description, dc.source, dc.language, dc.relation,
       dc.subject)
     * Types de données structurées : permet d’ajouter un support pour les
       données structurées afin d’obtenir une meilleure apparence dans les
       résultats de recherche Google.
     * Fil d’ariane : permet d’activer et d’utiliser un fil d’ariane
       optimisé avec le code suivant à insérer dans votre thème.

   [code]if(function_exists(‘seopress_display_breadcrumbs’)) { echo
   seopress_display_breadcrumbs(); }[/code]
     * Page Speed : permet d’intégrer Page Speed et ainsi vérifier les
       performances de votre site avec Google Page Speed.
     * Robots.txt : fonction utile qui permet de modifier rapidement le
       fichier robots.txt
     * Google News : l’ultime fonction ! qui permet d’ajouter un sitemap
       dédié pour Google News.
     * 404 : outil formidable afin de contrôler les erreurs 404 et de
       créer une véritable stratégie de redirection.
     * htaccess : fonction qui permet d’éditer facilement le fichier
       htaccess.
