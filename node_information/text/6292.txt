SCALABILITÉ DES APPLICATIONS AVEC NVIDIA NVLINK

   Pascal est la première architecture de GPU à intégrer la technologie
   d'interconnexion bidirectionnelle NVIDIA NVLink^™ à haute vitesse.
   Cette interface révolutionnaire permet de mettre à l’échelle les
   applications sur des GPU multiples avec un débit 5 fois plus élevé
   qu’avec les meilleures technologies actuelles.

   Architecture de GPU Pascal avec mémoire CoWoS pour Data Center

MÉMOIRE CoWoS AVEC HBM2 POUR LE BIG DATA

   L’architecture Pascal permet d’unifier le processeur et les données
   pour fournir une efficacité de calcul sans précédent. Grâce à sa
   conception innovante, la mémoire CoWoS^® (Chip-on-Wafer-on-Substrate) à
   technologie HBM2 délivre au moins 3 fois plus de performances par
   rapport aux solutions d'ancienne génération.
