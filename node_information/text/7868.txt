Documentation d'usage de nos headers/footers

Introduction

   Les headers et footers de notre site sont en accès libre via deux URLs
   qui retournent directement un bout de code HTML à intégrer de vôtre
   côté selon le besoin.
   Vous pouvez utiliser l'un sans l'autre mais certaines fonctionalités
   risquent de ne plus fonctionner (par exemple le menu de connexion du
   header).

Adresses

   L'adresse du header est la suivante :
   https://www.capital.fr/partials/header
   L'adresse du footer est la suivante :
   https://www.capital.fr/partials/footer

Exemple d'implémentation en PHP :

   Voici un exemple d'implémentation simple en PHP
<!doctype html>
<html lang="fr">
    <head>
        <meta charset="UTF-8" />
        <meta name="viewport" content="width=device-width, initial-scale=1">
    </head>
<?php
    echo file_get_contents("https://www.capital.fr/partials/header");
?>

<div class="bg-white">
    <div class="container bg-white m-top-md">
            Mon contenu
    </div>
</div>

<?php
    echo file_get_contents("https://www.capital.fr/partials/footer");
?>
</html>
